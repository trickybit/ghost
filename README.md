ghost
=====

LESS mix-in to the Netherworld

### Crosshatch
#### Input
```css
.ghost.crosshatch(blue);
```
#### Output
&#9641;

### Hatch
#### Input
```css
.ghost.hatch(blue, 1);
```
#### Output
&#9640;

#### Input
```css
.ghost.hatch(blue, -1);
```
#### Output
&#9639;

### Rule
#### Input
```css
.ghost.rule(blue, 0);
```
#### Output
&#9636;

#### Input
```css
.ghost.rule(blue, 1);
```

&#9637;
